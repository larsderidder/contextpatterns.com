The Problem

00.1

Context Rot

Model quality degrades as context gets longer, even well within the window limit. 11 of 12 models drop below 50% at 32k tokens. GPT-5 fails on reasoning tasks long before hitting its 272k limit. The window size on the box is not the window size you get. Every pattern below exists because of this.

Hong et al. (2025) · NoLiMa Benchmark
01

Patterns

01.1

The Pyramid

Start with general background, progressively add specific details. Give the model altitude before asking it to land. Context first, task second.

Anthropic
01.2

Select, Don't Dump

The smallest set of high-signal tokens that maximize the desired outcome. Surgical selection beats comprehensive inclusion. Every token is a cost.

NoLiMa · Chroma
01.3

Isolate

Give sub-agents their own focused contexts instead of sharing one massive window. 15x more tokens total, better results. Each agent sees only what it needs.

Anthropic
01.4

Compress & Restart

When conversations grow long, summarize what matters and start fresh. Compaction beats continuation. Treat context like a cache, not a log.

LangChain
01.5

Write Outside the Window

Persist important context to external storage. The context window is working memory, not long-term memory. What matters should survive beyond a session.

LangChain · Letta
01.6

Progressive Disclosure

Maintain lightweight identifiers, load full data on demand. Let the model discover what it needs through exploration, like navigating a codebase.

Anthropic
01.7

Recursive Delegation

Give the agent a code environment to programmatically manipulate its own context, and let it spawn child agents with controlled sub-contexts. Instead of stuffing 10M tokens into one window, the parent splits, delegates, and aggregates. Context becomes a tree, not a monolith.

Zhang et al. (2025)
